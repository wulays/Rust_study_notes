#### 元组

元组就是多种类型合到一起，同时元组的长度是固定的，元组中的顺序也是固定的，之前我们也见过了，空元组，也就是单元类型使用`()`包裹 `,` 分割来声明元组

```rust
let tup = (1, 's', "asdf");
```

可以使用 `.` 操作符来获取元组中的值，索引从 `0` 开始

```rust
let tup = (1, 'a', "bcd");

println!("{} {} {}", tup.0, tup.1, tup.2);
// 输出 1 a bcd
```

当然也可以去解构，同时也能显示声明类型

```rust
let tup: (i32, char, &str) = (1, 'a', "abcd");

let (x, y, z) = tup;

println!("{} {} {}", x, y, z);
// 输出 1 a abcd
```

一般也用来返回一组数据

```rust
fn main() {
	let (x, y) = get_xy();

	println!("{} {}", x, y);
}

fn get_xy() -> (i32, i32) {
	(10, 10)
}
```

这里就返回的是一对坐标，不过也不具有清晰的含义，毕竟两种取值方式也很通用，不能直接见名知意
